# 练迹 · FITLOG

一款为手机设计的轻量训练日志。无需注册，训练记录保存在浏览器本地；分享某天训练时，数据会编码进链接，对方打开即可查看。

## 功能

- 记录动作、组数、次数、重量、训练时长与感受
- 按胸、肩、背、腿、核心和有氧分类训练
- 在“分化数据库”中独立维护各部位的长期动作模板
- 在“总览作战台”中多选部位，将对应模板组合成当天训练
- 按部位使用独立霓虹色分组显示当天动作
- 复用历史动作模板与最近一次训练参数
- 记录动作要领、目标组、额外组和逐组完成进度
- 按月浏览历史训练
- 统计训练次数、时长、容量与常练动作
- 通过原生分享菜单或复制链接分享单日训练
- 移动端优先的响应式设计
- 支持安装到手机主屏幕并离线访问
- 可导出 JSON 格式的训练数据备份
- 深色赛博朋克界面，包含实时时钟、扫描线和周期脉冲动效

数据默认保存在当前浏览器中，不上传服务器。清理浏览器数据前建议先在“统计”页面导出备份。

## 本地开发

```bash
npm install
npm run dev
```
